Gaze documentation

This is the documentation for Gaze, a master's thesis project by Sebastian Höffner.

About and warning

Gaze is a library aimed at calibration free gaze tracking using normal webcams. However, it does not work properly. It might (but more likely might not) work in the future, but due to the nature of thesis projects I will most likely not pick it up anytime soon after I submitted my thesis.

If you are still interested in using it or developing it further, you should be able to find all relevant information in my thesis, the documentation, the source code, or by dropping me a line.

Final note

While the gaze tracking capabilities are very limited, the project still explores various exciting software developing practices:

  • Using Git for version control
  • Setting up GitHub and Semaphore CI for automated building, testing, and documentation deployment (this page you are viewing)
  • Modularized C++ code
  • "modern" CMake, i.e. 
    target_link_libraries(project gaze::gaze)
    is (almost) sufficient to use gaze
  • Example programs
  • Well documented
  • Free and Open Source Software
