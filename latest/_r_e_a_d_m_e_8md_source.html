42 ## Building
43 
44 ### Library
45 
46 Create a `build` directory and call `cmake`, followed by `make` to build the
47 gaze library:
48 
49 ```bash
50 mkdir
51 build
52 cd build
53 cmake ..
54 make
55 make install
56 ```
57 
58 
59 ### Demos
60 
61 To build the demo programs, you need to call cmake with `-DBUILD_EXAMPLES=ON`:
62 
63 ```bash
64 mkdir build
65 cd build
66 cmake -DBUILD_EXAMPLES=ON ..
67 make
68 make install
69 ```

71 
72 #### `where_people_look`
73 
74 This is a reimplementation of Judd et al. (2009)'s data acquisition experiment @cite Judd2009.
75 
76 It needs to be *installed* using `make install` to put the images into the
77 correct location. You can run it by simply typing `./where_people_look` from
78 the build directory.
79 
80 
81 #### `simple_tracking`
82 
83 This program just starts a small window with an "experiment" and tracks gaze.
84 Starts the gaze tracker in debug mode to see what is tracked.
