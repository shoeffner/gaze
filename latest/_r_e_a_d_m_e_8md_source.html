6 
7 Gaze aims at providing an easy to use gaze tracking library. It was the project for my master's thesis.
8 It is inspired by [eyeLike](https://github.com/trishume/eyeLike), and performs well in the task of eye tracking.
9 
10 *Warning:* Gaze tracking does not work!
11 
12 But you can try the `GazeCapture` pipeline step by modifying the `gaze.yaml` and configuring Gaze with `--caffe`. However, since the underlying CNN is trained on iPhone and iPad screens, it is most accurate in a smaller screen area around the camera.
13 
14 If you are interested in picking up the project or have some ideas on how to best fix the gaze tracking, make sure to drop me a line! [shoeffner@uos.de](mailto:shoeffner@uos.de)

51 ## Building
52 
53 To build gaze, simply run
54 
55 ```bash
56 ./configure.sh
57 cd build
58 make install
59 ```
60 
61 `configure.sh` comes with some options:
62 
63 - `--release`: Enables compiler optimizations
64 - `--debug`: Disables compiler optimizations, builds debug symbols and creates verbose makefiles.
65 - `--examples`: Configures cmake to create build files for the examples.
66 - `--doc`: Configures cmake to create build files for the documentation.
67 - `--tagfiles`: Links the documentation to other projects (OpenCV and the C++ standard library)
68 - `--dry`: Just prints the `cmake ..` command without executing it
69 - `--caffe`: Enables a build with Caffe support. This allows to use the experimental pipeline step `GazeCapture`.

72 ## Demo programs
73 
74 ### `where_people_look`
75 
76 This is a reimplementation of Judd et al. (2009)'s data acquisition experiment @cite Judd2009.
77 
78 It needs to be *installed* using `make install` to put the images into the
79 correct location. You can run it by simply typing `./where_people_look` from
80 the build directory.
81 
82 
83 ### `simple_tracking`
84 
85 This program just starts a small window with an "experiment" and tracks gaze.
86 Starts the gaze tracker in debug mode to see what is tracked.
